A Node JS - Express application that enables the user to:
- Upload single images.
- Upload multimple Images.
- Fetch random image.
- Fetch multiple random images.
- Fetch all images in the server.
- Fetch a paginated library of the images.

Technologies / Libraries used:
- Multer Middleware
- Handlebars Template Engine
- JQuery
- Bootstrap
- Popper
